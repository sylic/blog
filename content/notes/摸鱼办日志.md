我的摸鱼办日志

1. git 默认是忽略大小写的，文件名(文件夹)字母大小改变了，Git 不会跟踪到，关闭忽略大小写的命令如下：


   ```bash

   git config core.ignorecase false

   ```

2. 正则匹配，忽略大小写: ` new RegExp()` 第二个参数设为 `ig`

    ```
    var regular = new RegExp(`.*\.${suffix}`,"ig");
    ```
